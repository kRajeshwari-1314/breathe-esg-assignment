# Technical Decisions

## Django REST Framework

Chosen for:
- rapid API development
- serialization support
- scalable backend architecture

---

## React Frontend

Chosen because:
- lightweight UI development
- simple API integration
- component-based architecture

---

## SQLite

Chosen for:
- simplicity
- fast setup
- assignment suitability

Production systems can migrate to PostgreSQL.

---

## Pandas

Used for:
- CSV parsing
- data iteration
- schema handling

---

## Dynamic Schema Detection

Implemented using CSV column checks.

Benefits:
- supports multiple enterprise formats
- extensible ingestion pipeline
- cleaner architecture
